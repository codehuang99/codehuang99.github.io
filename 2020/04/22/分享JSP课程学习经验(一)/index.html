这篇是我在学习JSP课程的过程中自己根据课上内容总结出来的知识点，我只是自己想要分享下学习经验顺便供自己复习

1、JSP简介

①JSP全名是：Java Server Pages，用JSP开发的Web应用是跨平台的

②JSP与Servlet一样是在服务器端运行的，只要有浏览器就能浏览

③开发环境及开发工具：JDK、TomcatV8.5、MyEclipse2017

关于中文乱码问题：

一般出现乱码是因为三个原因造成的：文件编码≠页面指定编码≠浏览器所使用编码

  1. 文件编码：当你代码需要用到外部文件时候例如txt文件，则需要你在保存时候保存为utf-8格式

  2. 页面指定编码：

    1. 在你的jsp文件写入代码之前必须先指定页面的编码，如下图：utf-8格式

    2. 在编写java片段时候加上: request.setCharacterEncoding(“utf-8”)——–request请求时候设置编码;

    3. 在java片段里有中文输入时：<%@page import=”java.net.URLEncoder”%>和URLEncoder.encode(需要转换的,”utf-8”)

  3. 浏览器所使用编码(这里以myEclipse自带运行浏览器为例)：在运行后的界面右键-设置编码-utf8格式

2、JSP的基本语法

在jsp文件里引入java代码的三种方式：

# 第一种是引入声明变量、方法和类：
<%!
    # 声明变量，例如：
    int x=100;
    # 声明方法，例如：
    int add(int x,int y){
        return x+y;
    }
    # 声明类，例如：
    public class Circle{

    }
%>

# 第二种是引入表达式：
<%=Math.sin(0.9)/3 %>        # 相当于输出某个值 

# 第三种是引入java程序片段：
<%
    # 注意：不能写方法或者类
    int k=0;
    k++;

 %>

表单通过form标签的action属性来传递参数

# text文本框内容的获取：

5、JSP的指令和动作

Page指令

# 语法 <%@page 属性="内容" %> 例：
  <%@ page language="java" contentType="text/html;charset=UTF-8"%> 
#参数意义：
#    contentType="text/html";charaset="utf-8"    表示定义的字符编码和页面响应的类型
#    errorPage="error.jsp"                       表示页面出现错误时候要跳转错误的显示页
#    import="java.util.*";                          表示导入的java包
#    pageEncoding="utf-8"                        表示jsp页面的字符编码

include指令

# include表示静态插入，可以在jsp页面中插入其他类似文件，其他的jsp页面等，先包含一起再编译    
# 语法 <%@ include file= "文件的URL " %>  例：
  <%@ include file= "hello.txt" %>

include动作

# include动作是先编译原来jsp页面，当遇到include动作再包含进来编译    
# 语法 <jsp:include page= "文件的URL"/></jsp:include>   例：
  <jsp:include page= "index2.jsp"/></jsp:include>

forward动作

# forward动作是遇到时停止当前页面而转到其他页面    
# 语法  <jsp:useBean  id= “给bean起的名字” class= “创建bean的类" scope="bean有效范围"></jsp:useBean>     例：
  <jsp:forward page="num1.jsp">通过name然后获取到的是value值
      <jsp:param value="<%= n %>" name="item"/>
  </jsp:forward>

useBean动作

# useBean动作是可以获取到java类的值    
# 语法  <jsp:useBean id="对象名" class="类名" scope="作用域" />    例：
  <jsp:useBean id="lader" class="tom.bean.Lader" scope="page" />
# 参数含义：
#    id                                            定义该对象名称(自己取名字)
#    class                                       引入指定的java类名(与已有的java类名对应)
#    scope="page/request/session/application"    表示作用域范围，分别是：当前页面，请求过程，session会话期间、每个页面
#    pageEncoding="utf-8"                        表示jsp页面的字符编码

#  获取类的值,name对应的是useBean的id值，property对应的是类中定义变量名
<jsp:getProperty name="lader" property="height"  /> 

# 设置类的值,value是你要赋予的值
<jsp:setProperty name="lader" property="height" value="300" />

6、JSP内置对象

Page对象

# pageContext对象设置的属性只能在本页面中取得，无法在第二个页面获得    
# 语法  pageContext.setAttribute("属性名", "属性值");     例：
    <%
        # 设置一个属性为name，值为sun的
        pageContext.setAttribute("name", "sun");
        # 在本页面中获取该值-------因为是对象object要强制转换
        String refName = (String)pageContext.getAttribute("name"); 
    %>

Request对象

# request对象设置的属性可以通过<jsp：forward>传递在别的页面获取到，但通过a传递的获取不到    
# 语法  request.setAttribute("属性名", "属性值");     例：
    <%
        # 设置一个属性为name，值为sun的
        request.setAttribute("name", "sun");
        # 在本页面中获取该值-------因为是对象object要强制转换
        String refName = (String)request.getAttribute("name"); 
    %>

session对象

# session对象设置的属性可以通过<jsp：forward>或者a传递别的页面获得，除非退出浏览器换新浏览器打开
# session和cookie区别：session是在服务端的，随会话结束而关闭。而Cookie在存在客户端，长期保存在客户端     
# 语法  session.setAttribute("属性名", "属性值");     例：
    <%
        # 设置一个属性为name，值为sun的
        session.setAttribute("name", "sun");
        # 在本页面中获取该值-------因为是对象object要强制转换
        String refName = (String)session.getAttribute("name"); 
        # 移除session
        session.removeAttribute("设置的session属性名");
        session.invalidate();
    %>

Cookie对象

# cookie：Cookie在存在客户端，长期保存在客户端     
    <%
        # 创建cookie对象并传入值
        Cookie usernameCookie(自定义名)=new Cookie("cookie名(自定义)",传入的值)
        # 设置cookie生命时长----表示30分钟
        usernameCookie.setMaxAge(30*60);
        # 传给浏览器保存
        response.addCookie(usernameCookie)
        # 读取浏览器所有cookie，得到数组
        Cookie[] cookies=request.getCookies();
        # 读取cookie的键-------获取到的就是你设置的cookie名
        cookies[i].getName();
        # 获得cookie的值
        cookies[i].getValue();
    %>

Response对象

# 重定向:当重定向到别的页面时候，地址栏地址变成指定的页面地址，且在原来页面设置的内容获取不到，为null
# 语法  response.sendRedirect(地址);     例：
    <%
        String  url="regShow.jsp?msg="+msg;
        response.sendRedirect(url); 
    %>


# 转发:当进行转发时候，地址栏地址还是原来的页面地址，且转发后的页面可以获取到数据
# 语法  request.getRequestDispatcher(地址).forward(request, response);     例：
    <%
        String  url="regShow.jsp?msg="+msg;
        request.getRequestDispatcher(url).forward(request, response); 
    %>

调查问卷案例实现思路

  1. cookie：
    1. 在newsSelect页面先创建cookie对象，把对应的用户名传进去，设置生命时长，传给浏览器保存
    2. 在index获取cookie数组，先判断是否为空，遍历数组，获取name和value值，先判断name值是哪个进行输出并将对应的value值赋值给表单输入框的value值(这一步目的是：当我们输入用户名提交后再返回index页面，输入框内容会记住你之前的用户名并自动输入)

代码：

# index.jsp
<%@page import="java.net.URLEncoder"%>
<%@ page language="java" import="java.util.*" pageEncoding="UTF-8"%>

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">
<html>
  <head>
    <title>'index.jsp'</title>
  </head>

  <body>
    <div style="width:800px;margin:10px auto;lineheight:30px;">
        <h3>本浏览器已有该网站的Cookie列表</h3>
        <%
            String username="";
            String sex="";
            String sexChecked_0="",sexChecked_1="";
            Cookie[] cookies=request.getCookies(); # 读取浏览器的所有cookie，得到数组
            String name="";
            String value="";
            if(cookies!=null){//如果有cookie值
                for(int i=0;i<cookies.length;i++){ # 取出对应的名字和值
                    name=cookies[i].getName();
                    System.out.println(name);
                    System.out.println("-------------");
                    value=cookies[i].getValue();
                    System.out.println(value);
                    value=URLEncoder.encode(value,"UTF-8");# 并把值编码
                    out.print(name+"="+value+"<br>");# 输出所有的cookie值,也就是保存之前登陆过的信息
                    # 表示如果cookie值跟当前用户名一样，则把表单中的cookie的值输入给表单的value值
                    # 意思就是当你登陆完跳转到选择信息时候如果再点登陆页面他会自动在输入框写上登陆过的用户名和选中的性别
                    if(name.equals("username")){
                        username=value;

                    }else if(name.equals("sex")){
                        sex=value;

                    }

                }
                    if(sex.equals("")){
                        out.print("暂无性别信息<br>");
                    }
                    # 如果cookie的性别value值0则选择女，1选中男
                    if(sex.equals("0")){

                        sexChecked_0="checked='checked'";
                    }else if(sex.equals("1")){
                        sexChecked_1="checked='checked";
                    }

                    }
         %>
        <hr>
        <h3>欢迎参加网上调查，请先输入个人信息:</h3>
        <form action="newsSelect.jsp" method="post">
            昵称:<input type="text" name="username" value=<%=username %>>&emsp;&emsp;
            性别:<label><input type="radio" name="sex" value="0" <%=sexChecked_0 %>>女士</label>
                <input type="radio" name="sex" value="1" id="male" <%=sexChecked_1 %>><label for="male">先生</label>&emsp;&emsp;
            <input type="submit" value="提交" name="submit">

        </form>
        &emsp;<a href="./newsSelect.jsp">选择新闻栏目</a>&emsp;&emsp;
        <a href="./newsSelectShow.jsp">查看调查结果</a>
        <br>
        session默认在20分钟内有效，也可以设置有效期。浏览器关闭则失败。保存在服务器内存中。<br>
        cookie通常需设定有效期。保存在本地浏览器的临时文件夹中。
    </div>

  </body>
</html>


# newsSelect.jsp
<%@page import="java.net.URLEncoder"%>
<%@ page language="java" import="java.util.*" pageEncoding="UTF-8"%>


<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">
<html>
  <head>
    <title>'newsSelect.jsp'</title>
  </head>

  <body>
   <div  style="width:800px; margin:10px auto;lineheight:30px;">

   <%
    String username=request.getParameter("username");
    String sex=request.getParameter("sex");# 返回值是value
    String usernameSex=(String)session.getAttribute("usernameSex");# 从session读取值

    # 此情况是不通过表单提交而直接运行newselect页面才会出现null
    if(username==null && usernameSex==null){# 未提交个人信息，也无个人信息session
        response.sendRedirect("index.jsp");# 网页自动转向
        return;# 结束当前页面运行

    }

    if(username!=null){
        username=username.trim();
        if(username.equals("")){# 当通过表单提交但没有写入昵称时候此时，输入框获取到的值为空字符串
            out.print("请输入昵称:");
            out.print("<br><br><a href='javascript:window.history.back();'>后退</a>");
            return;

        }
        if(sex==null){# 输入框和单选按钮不一样，单选按钮没选就是null，而输入框没输入则为空字符串
            out.print("请选择性别:");
            out.print("<br><br><a href='javascript:window.history.back();'>后退</a>");
            return;
        }
        String sexShow="(性别未知)";

        if(sex.equals("0")){# 当性别获取到的value值为0，即为女生
            sexShow="女士";

        }else if(sex.equals("1")){# 当性别获取到的value值为1，即为男生
            sexShow="先生";

        }
        usernameSex=username+sexShow;# 把姓名和性别联系起来
        session.setAttribute("usernameSex", usernameSex);# 都传输到session

        String usernameEncode=URLEncoder.encode(username,"UTF-8");# 先把昵称编码
        Cookie usernameCookie=new Cookie("username",usernameEncode);# 创建cookie对象并传入昵称值
        usernameCookie.setMaxAge(30*60);# 设置生命时长为30分钟，超过时间信息失效
        response.addCookie(usernameCookie);# 把用户cookie传给浏览器保存

        Cookie sexCookie=new Cookie("sex",sex);# 创建cookie对象把性别也传入
        sexCookie.setMaxAge(30);
        response.addCookie(sexCookie);# 传给浏览器

    }
 %>

# newsSelectShow.jsp
<%@ page language="java" import="java.util.*" pageEncoding="UTF-8"%>
<%@page import="java.net.URLEncoder"%>

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">
<html>
  <head>
    <title>'newsSelectShow.jsp'</title>
  </head>

  <body>
     <div  style="width:800px; margin:10px auto;lineheight:30px;">
     <%
         request.setCharacterEncoding("utf-8");

         # 判断如果没有登录，也就是说你没登录就进入了查询结果页面就让你返回index界面
         String usernameSex=(String)session.getAttribute("usernameSex");# 从session读取值
         if(usernameSex==null){
             response.sendRedirect("./index.jsp");
             return;

         }

         String news[]=request.getParameterValues("news");
         String msg="";
         if(news==null){
             msg="您什么都没有选择";

         }else{
             for(int i=0;i<news.length;i++){
                 msg+=news[i]+"<br>";
             }

         }

         Integer count=(Integer)session.getAttribute("count");# 选择的轮数
         if(count==null){# 如果没有参加调查，则为0
             count=0;

         }
         if(request.getParameter("submit")!=null){
             count++;
             session.setAttribute("count", count);
         }
         msg+="<br>这是您第【"+count+"】轮选择";
      %>

         【<%=usernameSex %>】,欢迎参与调查&emsp;(<a href="./logout.jsp">退出个人登录</a>)
